Instalação de Sistemas Operacionais

Como instalar o Windows pelo pendrive bootável

Com a popularização de notebooks sem drive de CD/DVD, surgiu uma nova necessidade: instalar sistemas operacionais sem depender do uso de mídias físicas em disco, já que elas não poderiam ser lidas nos computadores mais finos e super compactos. Como todos eles têm ao menos uma porta USB sobrando, esse virou o método preferido. Embora o alvo mesmo sejam os usuários sem drive de CD, os procedimentos a seguir podem ser usados por qualquer pessoa em qualquer computador que tenha a capacidade de fazer boot usando as suas portas USB.

O que você vai precisar:


● Pen drive USB com pelo menos 4GB;
● CD ou DVD de instalação do seu sistema operacional escolhido (Windows XP*, Windows Vista ou Windows 7 etc);
● O programa Novicorp WinToFlash,o download é gratuito;
● Um computador que já tenha alguma versão do Windows rodando.

Passo a Passo:

1. Baixe e descompacte o WinToFlash. Ele não vai criar uma pasta a menos que você configure isso na hora da extração. Portanto, crie uma pasta antes e evite que os ícones do programa fiquem voando soltos pela sua área de trabalho


2. Insira o CD/DVD do sistema operacional no drive e execute o WinToFlash. Ele vai oferecer duas opções: Setup Wizard (simples) e uma avançada;


3. Se você não é um usuário experiente, vá pela simples, pois com ela não há como dar errado. A parte mais complicada do processo consiste em clicar no botão Next várias vezes. O programa vai perguntar qual é o drive do CD/DVD e qual é o drive do pendrive USB.


4. Depois disso, ele deverá detectar qual é o CD e qual detectar qual o sistema no disco e mostrar a EULA (Termos de Uso) do sistema que deve ser aceita para prosseguir. Depois desse passo, o processo de configuração (formatação e cópia dos arquivos do CD/DVD) do pendrive USB será iniciado;


5. Caso você opte pelo modo avançado, o programa oferece algumas opções extras, como a instalação de um Windows PE (Pre-Installed Environment), criação de um pendrive USB de boot de emergência (para quando acontecem erros como Falta NTLDR, por exemplo) entre outras opções (bastante úteis por sinal);


6. Uma vez que o sistema esteja no pendrive, vá para o computador no qual ele será instalado. Ao ligá-lo, entre no setup da BIOS. A tecla necessária para abrir o setup varia de fabricante para fabricante, mas normalmente é uma das três: DEL, F2 ou F9. Só para garantir, aperte todas;


7. Uma vez na BIOS, encontre a sessão de Boot e coloque o Removable USB Disk como a primeira opção. Isso vai fazer com que o computador busque os arquivos de instalação ou boot em alguma das portas USB. Salve as configurações e saia do Setup.


8. Insira o pendrive logo depois de sair do Setup. A tela de instalação deverá aparecer.


9. Selecione a primeira opção e instale os arquivos iniciais. Quando o programa de instalação reiniciar a máquina, selecione a segunda opção e continue instalando.


Observações

Esteja atento:

● Não é recomendado usar uma ISO de instalação montada em um driver de CD virtual para criar o pendrive. Aumenta a chance de erros durante a cópia.

Como fazer o boot com CD no Windows

Ao inicializar o Windows com o CD de instalação, é possível resolver alguns problemas de funcionamento do sistema e até mesmo instalar o Windows.

Para fazer o boot com o CD de instalação no Windows, insira o CD no leitor e reinicie o computador. Quando ele aparecer no monitor, o que acontece rapidamente, pressione várias vezes a tecla F8 até que o menu de Inicialização seja exibido. Nesse menu, selecione o CD-ROM de instalação. Um convite aparecerá no monitor para executar o CD. Para rodá-lo, aperte qualquer tecla do seu teclado. Não deixe de pressionar a tecla F8, senão o PC inicia sem o CD.

A partir daí, siga os passos solicitados pelo CD de instalação com muita atenção.

Como baixar Windows 10 e instalar no seu PC

Para resolver a necessidade de uma mídia para baixar Windows 10 e instalar, como um DVD ou pendrive, a Microsoft criou uma ferramenta especial, o "Media Content Tool", a qual o usuário pode baixar e então utilizá-la para instalar Windows 10 ou criar uma mídia para instalá-lo.


Passo 1. Acesse este link oficial (https://www.microsoft.com/pt-br/software-download/windows10) da Microsoft e clique em "Baixar a ferramenta agora" para baixar o Windows 10. Acesse o site oficial da Microsoft para fazer download da ferramenta oficial e baixar Windows 10 para realizar a instalação

Passo 2. Abra a ferramenta, leia os termos de licença para software Microsoft e depois clique em "Aceitar". Concorde com os termos de licença da Microsoft para baixar Windows 10 e prosseguir com a instalação

Passo 3. Caso seja este mesmo computador ou notebook que baixou a ferramenta no qual pretendia instalar o Windows, clique em "Atualizar este computador agora" e "Avançar". Se deseja instalar em outro computador, falaremos mais abaixo. Caso a instalação Windows 10 esteja neste mesmo PC, selecione a opção de atualizar. Caso contrário, veja mais abaixo

Passo 4. Aguarde o software baixar Windows 10 para dar prosseguimento à instalação. Aguarde o software da Microsoft baixar Windows 10 para continuar a instalação

Passo 5. Algumas etapas extras serão realizadas e então a instalação perguntará se o usuário deseja manter seus aplicativos e arquivos pessoais, apenas arquivos pessoais ou se não pretende manter nada. Às vezes não é possível escolher manter arquivos e neste caso os dados serão formatados, então faça backup antes. Após escolher, selecione "Avançar". Em alguns casos, não será possível manter os arquivos antes de instalar Windows 10, então faça backup antes

Passo 6. Após todos os preparativos, confira as opções escolhidas e clique em "Instalar" para começar a instalar Windows 10. Após baixar Windows 10 e selecionar todas as opções de instalação, confirme no botão "Instalar"

Passo 7. A seguir, a instalação Windows 10 terá início e o PC será reiniciado várias vezes. O processo pode ser bem demorado. Aguarde a instalação Windows 10 ser terminada, algo que pode demorar um pouco

Passo 8. Quando a instalação Windows 10 estiver quase terminada, o usuário precisará decidir alguns detalhes nas configurações, como região, layout do teclado, uso de localização por apps, entre outros. Na etapa final da instalação Windows 10, o usuário precisará decidir detalhes do sistema operacional como idioma e layout de teclado

Passo 9. Nesta etapa também é possível fazer login em sua conta Microsoft ou criar uma nova. Perto do final da instalação Windows 10 será possível também fazer login em sua conta da Microsoft

Passo 10. Ao final de tudo, seu Windows 10 estará instalado e pronto para ser usado.
